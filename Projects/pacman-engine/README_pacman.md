# 🎮 Pac-Man Engine — Python

Recréation complète du moteur de jeu **Pac-Man** en Python avec Pygame.  
Implémentation from scratch des mécaniques de jeu, des collisions et des algorithmes de comportement des fantômes.

---

## Fonctionnalités

- Moteur de déplacement du joueur (contrôles clavier)
- Algorithmes de comportement des fantômes (poursuite, fuite, aléatoire)
- Détection de collisions complète (murs, fantômes, pac-gommes)
- Gestion du score et des niveaux
- Code structuré en **programmation orientée objet**

---

## 🚀 Installation & lancement

```bash
# 1. Cloner le repo
git clone https://github.com/jp-jreissaty/pacman-engine.git
cd pacman-engine

# 2. Installer les dépendances
pip install pygame

# 3. Lancer le jeu
python main.py
```
